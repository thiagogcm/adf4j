# Support

Use GitHub issues for support requests, bug reports, and feature requests:

<https://github.com/thiagogcm/adf4j/issues>

When reporting a Chrome extension issue, include:

- Chrome version and operating system.
- The extension version from `chrome://extensions`.
- Whether the page URL matches `https://*.atlassian.net/wiki/*`.
- The visible button state or error title.
- Reproduction steps that do not include private Confluence content.

## Security vulnerabilities

Please do not report security vulnerabilities in public issues.

Use GitHub's private advisory flow:

<https://github.com/thiagogcm/adf4j/security/advisories/new>

If you cannot use that form, follow the fallback contact path in the repository security policy:

<https://github.com/thiagogcm/adf4j/blob/main/SECURITY.md>
